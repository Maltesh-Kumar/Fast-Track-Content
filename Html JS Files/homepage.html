Content Modelling Intelligence

What Does CMI Do?

Nowadays within the period of having huge information, literary information is rapidly developing and is accessible in numerous diverse languages. Often due to time limitations we are not able to devour all the information that is accessible. With the fast-moving world, it’s troublesome to peruse all the text-content. Hence, the necessity for content summarization comes to the spotlight. It is in this manner we are able to summarize the content so that it gets easier to ingest the data, keeping up the substance and understanding of the data

Content Modelling/Automatic Text Summarization is one of the foremost challenging and serious issues within the field of Natural language processing (NLP). It could be prepared by producing a brief and important outline of content from different content assets such as raw text, news api, images and document summarization.


Our software model consists of content summarization methods for various Indian and foreign languages like Hindi, Kannada, Telugu, Marathi, French, German etc. It also encompasses a system of Optical Character Recognition (OCR) which extracts the content from the given picture. The main motive of an OCR is to create editable records from existing paper records or picture files. An OCR also works on sentence discovery to protect a document’s structure. It also contains a text-to-speech component that will provide an mp3 audio of the summary in the chosen language.

Input Options

Raw Text.

Raw text input that is given by the user is summarized.

User needs to chose the desired language for summary. The data to be summarized can be pasted into the text box to obtain modelled content on submission. An audio[mp3] file of the summary in the selected language is also accessible for download.


CLick the button below to proceed!


Image Data.

JPEG Image input that is given by the user is summarized.

User must choose and upload the image of their choice and select the desired language for summary. The data in the image is then read using Optical Character Recognition(OCR) and then modelled and displayed as output. An audio[mp3] file of the summary in the selected language is also accessible for download.


CLick the button below to proceed!


News API.

Sample news APIs covering multiple domains can be summarized.

User must select the desired language for summary. The user can opt for a news article from the given list and press the submit arrow on the desired article to obtain the modelled content. An audio[mp3] file of the summary in the selected language is also accessible for download.


CLick the button below to proceed!


Document.

A document uploaded by the user can be summarized.

User must choose and upload the docuemnt of their choice and select the desired language for summary. The data in the document is then modelled and displayed as output. An audio[mp3] file of the summary in the selected language is also accessible for download.
